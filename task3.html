Task 3: Fixed vs Flexible Columns — Answers

1. Which layout breaks faster on small screens?
Fixed-width (px) layout.
2. Which layout adjusts automatically?
Flexible (fr) layout.
3. Why is fixed layout risky?
Fixed pixel sizes don’t change with screen size, causing overflow.
4. Where can fixed columns be useful?
Dashboards, tables, or designs needing exact sizing.
5. Which unit for a responsive website?
The fr unit.
