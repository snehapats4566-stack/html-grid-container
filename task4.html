1. What does auto do in grid rows?
Auto adjusts the row height according to content.
2. Which row grows based on content?
The row that has the auto value.
3. Can rows have different heights?
Yes, each grid row can have a different height.
4. What happens if content is very large?
With auto, the row expands. With fixed height, content overflows.
5. Where are grid rows useful in real layouts?
For page sections like headers, content areas, and footers.
