Task 8: Auto-fit, Minmax & Responsive Grid

What problem does auto-fit solve?

Auto-fit automatically adjusts the number of columns based on available space.
Why is minmax important?

Minmax sets a minimum and maximum size so items do not become too small or too large.
Do items shrink too much?

No, minmax prevents items from shrinking below the minimum size.
Is this layout mobile friendly?

Yes, the layout adjusts automatically on different screen sizes.
Why are media queries not needed here?

Because Grid handles responsiveness automatically using auto-fit and minmax.
