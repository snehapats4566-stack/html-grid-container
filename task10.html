Task 10: CSS @supports

Why do we check browser support?

To ensure the browser supports CSS Grid before using it.
What happens if Grid is not supported?

The Grid styles are ignored and normal layout is used.
Is this needed for all projects?

No, it is mainly useful for older browsers.
Who uses this in real projects?

Professional developers use it for better looking layouts.
What does safe CSS mean?

Safe CSS means styles that work across different browsers.
